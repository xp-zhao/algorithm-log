## 定长滑动窗口

## 题目
- [LeetCode 1456 定长子串中元音的最大数目](../java/src/practice/01_array_string/MaximumOfVowelsInSubstring.java)

若窗口长度为 `k` , 当窗口右端点在 `R` 时，窗口左端点在 `R-k+1`

> 对于窗口 [L, R], 里面元素个数为 `R-L+1`. 例如 [2, 2] 中一个有 `2 - 2 + 1 = 1` 个元素。如果窗口大小为 `k` , 即 `R - L + 1 = k`, 则 `L = R - k + 1`

滑动窗口的操作分为三步：**入-更新-出**
1. **入：** 下标为 `l` 的元素进入窗口，更新相关统计量。如果此时窗口左端点 `l - k + 1 < 0` 说明窗口未满，重复第一步
2. **更新：** 更新答案。一般是更新最大值/最小值
3. **出：** 下标为 `l - k + 1` 的元素离开窗口，更新相关统计量
# 解题模板

定长子串中元音的最大数目
```Java
class Solution{
	public int maxVowels(String str, int k){
		char[] s = str.toCharArray();
		int count = 0;
		int vowel = 0;
		// 循环窗口右端点 r
		for (int r = 0; r < s.length; r ++){
			// 1. 右端点进入窗口
			if(isVowel(s[r])){
				vowel++;
			}
			// 左端点
			int l = r - k + 1;
			if(r < 0){
				// 窗口未满，重复第一步
				continue;
			}
			// 2. 更新答案（当前元音的最大数目）
			count = Math.max(count, vowel);
			
			// 优化点：如果答案已经达到理论上的最大值，可直接返回
			if(count == k){
				return count;
			}
			
			// 3. 左端点离开窗口，未下一个元素进入窗口做准备
			char out = s[l];
			if(isVowel(out)){
				vowel--;
			}
		}
		return count;
	}
	
	public boolean isVowel(char c){
		return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
	}
}
```
